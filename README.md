# 🍽 Smart Food Waste Monitoring System  
### AI-Enabled Canteen Food Management Application  
Built using Zoho Creator

---

## 📖 Project Overview

The Smart Food Waste Monitoring System is a low-code application developed using Zoho Creator to monitor, analyze, and reduce food wastage in college and public canteens.

This system tracks daily food preparation, serving, and waste quantities, generates analytics dashboards, and provides intelligent insights to optimize food production and reduce financial and environmental losses.

---

## 🚨 Problem Statement

In many colleges and public food facilities:

- Large quantities of food are wasted daily
- No structured data tracking system exists
- Overproduction leads to financial loss
- No predictive system to estimate optimal preparation quantity

This project solves these problems through data tracking, analytics, and AI-based insights.

---

## 🎯 Objectives

- Track daily food preparation and waste
- Calculate waste percentage automatically
- Monitor supplier inventory and cost
- Generate dashboards for analysis
- Enable role-based access (Admin / Staff)
- Reduce overproduction using data-driven insights

---

## 🏗 Application Modules

### 1️⃣ Dashboard
- Average Quantity Prepared
- Average Quantity Served
- Average Quantity Wasted
- Total Costs
- Meal-wise statistics
- Graphical analytics

### 2️⃣ Daily Food Entries
- Date
- Meal Type (Breakfast / Lunch / Dinner)
- Food Item
- Quantity Prepared
- Quantity Served
- Quantity Wasted (Auto-calculated)
- Remarks

**Formula Used:**
Quantity Wasted = Quantity Prepared - Quantity Served

---

### 3️⃣ Supplier Inventory
- Supplier Name
- Item Supplied
- Quantity Supplied
- Cost
- Supply Date

Used to track raw material cost and financial impact of food waste.

---

### 4️⃣ Meal Types Management
- Breakfast
- Lunch
- Dinner

Used for categorized analytics.

---

### 5️⃣ User Roles
- Admin
- Canteen Staff

Role-based access control implemented.

---

### 6️⃣ Daily Waste Summary
- Total Prepared
- Total Served
- Total Wasted
- Waste Percentage
- Entry Count

**Waste Percentage Formula:**
Waste % = (Total Wasted / Total Prepared) × 100
